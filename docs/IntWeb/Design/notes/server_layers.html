ZMap Server layering

Orientation

ZMap implements 'servers' whcih provide access to databases either via an interactive connection (eg ACEDB) or a passive one (eg pipeServers) and a few layers of software have been implemented to allow this. These layers have thoer own priovate data and information is passed bewteen the layers via call back functions.
