ZMap has a fairly complex user interface and this guides describes its major aspects.

All Windows

Short Cuts

The following short cuts can be used on all zmap windows:

Short Cut Action
Cntl-W Close this window
Cntl-Q Quit ZMap

Feature Display Window

Short Cuts

Short Cut Action
+ (or =), - zoom in/out a bit (XXX)
Cntl + (or =), Cntl - zoom in a lot (XXXX)
up-arrow, down-arrow scroll up/down a bit
Cntl up-arrow, Cntl down-arrow scroll up/down a bit
left-arrow, right-arrow scroll left/right a bit
page-up, page-down page up/down by XXXX
Cntl page-up, Cntl page-down page up/down by XXXX
