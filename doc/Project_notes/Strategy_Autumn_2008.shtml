Overview

We have been bug fixing and bug fixing but now we need to make some larger changes again.

Goals

Improving Testing

ZMap could be tested by adding code so that zmap would run off GFF and styles files. This could be done in batch and commands sent to zmap from Roys X Atoms/Properties interface tester. Zmap could also dump data so allowing diff'ing of output files to catch changes in data positions/types etc.

The following pieces of code would be required to do this:

  • The GFF file reader code would need to be revamped.

  • The Styles file reader would need to be revamped.

  • The GFF dumper would need to be revamped.

  • We should add a complete context dumper (including styles).

  • Roys command program should be able to take a file containing a list of commands and pass them over.

  • We should also diff the zmap log file.
